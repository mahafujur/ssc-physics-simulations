# শিখো ফিজিক্স ল্যাব ⚛️💚
### SSC Physics 3D Simulation Lab — শিখবো, জিতবো!

এনসিটিবি **পদার্থবিজ্ঞান (নবম-দশম শ্রেণি)** বইয়ের ১৪টি অধ্যায়ের প্রতিটির জন্য
**বাস্তব-দৃশ্যের**, টাচ-ফ্রেন্ডলি ৩ডি সিমুলেশন — সম্পূর্ণ বাংলায়, শিখো-থিমে।

## চালানোর নিয়ম (How to run)

```bash
npm install
npm run dev        # http://localhost:3000
```

প্রোডাকশন: `npm run build && npm start`

## ফিচার
- 💚 শিখো ব্র্যান্ড থিম (সবুজ + হলুদ + গাঢ় সবুজ-কালো)
- 🪐 ইন্টারঅ্যাকটিভ ৩ডি ল্যান্ডিং: ১৪টি অধ্যায়-গ্রহ ঘুরছে — **গ্রহে ট্যাপ করলেই সেই অধ্যায়ে!**
  পরমাণুতে ট্যাপ = কণা-বিস্ফোরণ + শকওয়েভ রিপল
- 🎮 লেভেল-সিলেক্ট অধ্যায় গ্রিড, প্রতিটির পূর্ণ NCTB টপিক তালিকা (১১০+)
- 👆 কাস্টম টাচ-অরবিট: এক আঙুলে ঘোরাও, দুই আঙুলে পিঞ্চ-জুম, ট্যাপ = ইন্টারঅ্যাকশন
- 🌊 "৪ডি" ফ্লুইড জ্যামিতি: পানির ঢেউ, জীবন্ত আগুন, মর্ফিং নিউক্লিয়াস

## ১৪টি বাস্তব-দৃশ্য সিমুলেশন
| অধ্যায় | দৃশ্য |
|---|---|
| ১. পরিমাপ | ক্লাসরুমের টেবিলে উপহার-বাক্স + স্কেল + ব্ল্যাকবোর্ড |
| ২. গতি | ⚽ ফুটবল মাঠ: কিকার, ঘূর্ণায়মান বল, গোলপোস্ট = পাল্লা |
| ৩. বল | 📦 রাস্তায় ক্রেট ঠেলা — ঠেলাওয়ালা সহ, F = ma |
| ৪. শক্তি | 🛝 দোলনায় বাচ্চা + শক্তি-টাওয়ার (PE/KE) |
| ৫. চাপ | ⛵ পুকুর: ঢেউ, মাছ, নৌকা ভাসে / পাথর ডোবে |
| ৬. তাপ | 🔥 চুলার জীবন্ত আগুনে দণ্ড — লাল হয়ে প্রসারিত, বাষ্প |
| ৭. তরঙ্গ | 🌅 সন্ধ্যার মাঠে রঙিন দড়ি-তরঙ্গ + স্পিকারের শব্দ-রিং |
| ৮. প্রতিফলন | 🔦 সোনালি ফ্রেমের আয়নায় টর্চ + ছুটন্ত ফোটন |
| ৯. প্রতিসরণ | ✏️ গ্লাসের পানিতে বাঁকা পেন্সিল (স্নেলের সূত্রে কিঙ্ক!) |
| ১০. স্থির বিদ্যুৎ | 🎈 দুটি চার্জড বেলুন + লাফানো কাগজের টুকরো |
| ১১. চল বিদ্যুৎ | 💡 রাতের ঘরে ঝুলন্ত বাতি — দেয়াল আলোকিত হয় |
| ১২. চৌম্বক | 🏗️ বৈদ্যুতিক চুম্বক ক্রেন — পেরেক তুলে নেয়! |
| ১৩. আধুনিক | ⚛️ মহাজাগতিক পরমাণু: নীহারিকা, ইলেকট্রন-ট্রেইল |
| ১৪. জীবন বাঁচাতে | 🏥 হাসপাতাল: রোগী, স্পন্দিত হৃদয়, ইসিজি মনিটর |

## স্ট্রাকচার
- `data/chapters.js` — NCTB অধ্যায়/টপিক ডেটা
- `lib/sims.js` — পরিবেশ-নির্মাতা + ১৪টি সিন (SIM_DEFS)
- `components/Sim3D.jsx` — three.js ইঞ্জিন (টাচ অরবিট + ট্যাপ রেকাস্ট)
- `components/SimView.jsx` — স্লাইডার/রিডআউট প্যানেল
- `app/` — রুট: `/`, `/chapters`, `/chapter/[id]`, `/sim/[id]`

নতুন সিম যোগ করতে: `lib/sims.js`-এ SIM_DEFS-এ এন্ট্রি লেখো → `data/chapters.js`-এ `sim:` কী মেলাও।
